Tim Berners-Lee: The Inventor of the World Wide Web

Here are the three main components of the World Wide Web as invented by Tim Berners-Lee:

HTML (HyperText Markup Language)
The language used to create and structure content on the web.
HTTP (HyperText Transfer Protocol)
The protocol that allows communication between web browsers and servers.
URI (Uniform Resource Identifier)
A system of unique addresses for identifying resources on the web.
